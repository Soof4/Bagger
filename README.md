# Bagger
A TShock plugin that let's players get bags for the bosses they've missed. These bosses are:
* King Slime
* Eye of Cthulhu
* Eater of Worlds / Brain of Cthulhu
* Skeletron
* Wall of Flesh
* The Destroyer
* The Twins
* Skeletron Prime
* Plantera
* Golem

## Permissions and Commands
|Permissions  | Commands    |
|-------------|-------------|
|bagger.getbags |getbags, gb|
